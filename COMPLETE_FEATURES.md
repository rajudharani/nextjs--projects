## 🚀 **How to Run:**

1. **Start Database:**
   ```bash
   npx json-server --watch src/app/_data/db.json --port 3005
   ```

2. **Start App:**
   ```bash
   npm run dev
   ```

3. **Open Browser:**
   Go to `http://localhost:3000`
